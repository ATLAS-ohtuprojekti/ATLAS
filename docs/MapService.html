Class: MapService

MapService(atlasMap, gridArray) → {MapService}

new MapService(atlasMap, gridArray) → {MapService}

Provides an interface for map-related functionalities. When a ready-made atlas map is given as an argument, it is used as it stands as an archetypal atlas map. Otherwise, grid data must be given as an argument for a base map and grid overlay to be created during construction. The base map is created using specific GeoJSON files.
Parameters:
Name Type Description
atlasMap Object An archetypal atlas map that contains certain mandatory elements. It can be an SVG XMLDocument (https://developer.mozilla.org/en-US/docs/Web/API/XMLDocument) or an SVG string.
gridArray Array An object array containing grid objects with an id as well as north and east coordinates in Finland Uniform Coordinate System (EPSG:2393).
